Hepatitis C virus (HCV) genotyping at our institution is performed using the Versant Hepatitis C virus genotype 2.0 Line Probe Assay (LiPA). HCV Genie 2 utilizes a scanned image to generate the genotypes rather than a manual, laborious, and error prone process that involves the comparison of the banding pattern on a test strip to a physical reference table.

HCV Genie 2 is built entirely in client side JavaScript, this means that your data will never leave your computuer. Instead the image analysis tools migrate to your browser and provide you with the genotyping result. It is due to this that we claim HIPAA compliance. It should be noted that individual institutions may have different rules in regards to HIPAA compliance, so you should check with your instituion before utilizing this tool.

HCV Genie 2's paper has been submitted and a link will be provide here for more information when it become avaliable.

The procedure we recommend for decreasing processing time in your laboratory is as follows:

  1. Run the LiPA analysis as normal.
  2. While still wet, place test strips on a laminated beige piece of paper. (Download Example)
  3. Pat the strips and page dry with disposable towel.
  4. Scan to a PDF face down on a flatbed scanner, then drag the file into this tool.
